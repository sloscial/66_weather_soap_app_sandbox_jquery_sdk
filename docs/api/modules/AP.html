AP Module

Defined in: sdk/utility/validator.coffee:1

Provides the global namespace for SDK framework classes. Provides convenience methods for app management.

Example:

class AppClass extends AP.Application

    Unlike a normal AP.collection.Collection, aggregate collections expect a simple integer-only response from the server. Aggregate collections are always zero-length (they have no members). They have an extra member value.

    Mock server instances are used by the test framework to intercept XHR requests and simulate the functionality of an API backend, entirely offline.     A belongs-to relationship is one where the owner model instance is related to no more than one other model instance. In this scheme, the relationship information is stored in a foreign key on the owner model. The related instance is stored in a generated field {@link #name} once fetched.

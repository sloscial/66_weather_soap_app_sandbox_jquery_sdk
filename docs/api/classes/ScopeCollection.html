Defined in sdk/collection/base_scope_collection.coffee:25

Copies data to new object and replaces keys which match any queryFields mapping configurations with the alternative parameter name. For example, with queryFields [{fieldName: 'name', paramName: 'person_name'}] and input object{name: 'John', age: 35}, output object is{person_name: 'John', age: 35}`.

    name/value pairs to map

Authorization Class

Defined in: sdk/auth/authorization.coffee:4
Module: auth
Parent Module: AP

Authorizes arbitrary objects against the currently logged-in user (see AP.auth.Authentication). Any object may be made permission-based by adding an auth rules field. If the currently logged-in user has any role specified in the rules array, it is considered authorized.
